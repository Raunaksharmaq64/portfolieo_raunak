## ✨ Features & Visual Highlights

* **Solid Color Obsidian Theme:** Overhauled layout matching gold-amber highlights with dark obsidian cards for a custom, bespoke designer look.
* **Interactive 3D Card Hover:** Handcrafted 3D card tilt transformations (`TiltCard`) that respond smoothly to cursor coordinates and restore to resting angles.
* **Interactive Developer Console (CLI):**
  * A custom-built floating terminal console supporting tab auto-completion and key history parsing.
  * Interactive shell commands: `neofetch` (stats), `projects` (detailed project directory), `theme` (real-time color toggle), `locate <sec>` (smooth page scrolling), `resume` (PDF downloads), and `cls` (clear screen).
* **Direct Email Contact Handling:** Controlled React form parsing that forwards enquiries directly to my mail inbox using Web3Forms endpoints.
* **Automated CI/CD Pipeline:** Fully configured GitHub Actions workflow (`deploy.yml`) that auto-compiles and deploys changes to GitHub Pages on every git push.

## 🚀 Local Installation & Developer Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Raunaksharmaq64/raunak.git
   cd raunak
   ```

2. **Install node dependencies:**
   ```bash
   npm install
   ```

3. **Run the local Vite development server:**
   ```bash
   npm run dev
   ```

4. **Compile the production bundle:**
   ```bash
   npm run build
   ```
